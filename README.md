# Smart-Watch-Health-Monitor
UI/UX case study of a smartwatch app with a voice assistant for chronic disease monitoring
An interactive UI/UX design and research case study for a smartwatch application integrated with a voice assistant, specifically tailored for chronic disease patients (Diabetes & High Blood Pressure).
Problem Statement
Many elderly and chronic disease patients face difficulties managing multiple separate medical devices (glucometers and sphygmomanometers) to monitor their daily health. This project solves this by integrating these vital tracking features into a single, highly accessible smartwatch application equipped with a voice assistant to assist with tasks, remind users of medication, and read health metrics aloud.
Research & Methodology
User Survey: Conducted a comprehensive questionnaire for respondents to validate user needs, challenges, and core struggles (such as forgetting medication, lack of activity tracking, and complex sync issues).
User Interviews: Gathered qualitative feedback through direct interviews with the target group to refine app features, confirming a strong demand (97.7%) for device integration.
UI/UX Design Process
1. Sketches: Created initial hand-drawn concepts to map out core navigation and user flows.
2. Wireframes: Developed low-fidelity layouts customized specifically for smartwatches.
3. High-Fidelity Prototypes: Designed the final polished screens, incorporating dynamic transitions and interactive on-drag components.
